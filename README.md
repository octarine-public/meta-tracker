# meta-tracker

Скрипт для [Octarine](https://github.com/octarine-public/wrapper): показывает винрейт, пикрейт и тирлист героев прямо на сетке героев в Dota 2 (в лобби и в игре) и на странице героев в дашборде.

## Возможности

- **Оверлей статистики** — на карточках героев отображаются винрейт, пикрейт и тир (S/A/B/C/D).
- **Два источника данных** — официальные данные Dota 2 или [Stratz](https://stratz.com).
- **Stratz**: фильтры по рангу (Herald–Immortal), позиции (Carry–Hard support) и периоду (день / неделя).
- **Dota 2**: выбор ранга для отображения статистики.
- **Легенда тиров** — панель на странице героев в дашборде с расшифровкой тиров.
- **Панель настроек** в дашборде для переключения источника и фильтров без выхода из клиента.

Меню: **Visual → Meta tracker** (вкл/выкл, тирлист, источник Stats, при Stratz — Rank, Period, Position).

# meta-tracker

[Octarine](https://github.com/octarine-public/wrapper) script that shows hero win rate, pick rate, and tier list on the hero grid in Dota 2 (lobby and in-game) and on the dashboard heroes page.

## Features

- **Stats overlay** — win rate, pick rate, and tier (S/A/B/C/D) shown on hero cards.
- **Two data sources** — official Dota 2 data or [Stratz](https://stratz.com).
- **Stratz**: filters by rank (Herald–Immortal), position (Carry–Hard support), and period (day / week).
- **Dota 2**: rank selection for displayed stats.
- **Tier legend** — panel on the dashboard heroes page explaining tier grades.
- **Settings panel** in the dashboard to switch source and filters without leaving the client.

Menu: **Visual → Meta tracker** (enable/disable, tier list, Stats source; with Stratz: Rank, Period, Position).

# meta-tracker

[Octarine](https://github.com/octarine-public/wrapper) 用脚本：在 Dota 2 英雄网格（大厅与对局内）以及仪表盘英雄页面上显示英雄胜率、选取率与梯队。

## 功能

- **统计叠加** — 在英雄卡片上显示胜率、选取率与梯队（S/A/B/C/D）。
- **双数据源** — 使用 Dota 2 官方数据或 [Stratz](https://stratz.com)。
- **Stratz**：按段位（Herald–Immortal）、位置（Carry–Hard support）与周期（日/周）筛选。
- **Dota 2**：选择段位以显示对应统计。
- **梯队图例** — 仪表盘英雄页面上的梯队说明面板。
- **设置面板** — 在仪表盘中切换数据源与筛选，无需退出客户端。

菜单：**Visual → Meta tracker**（开关、梯队列表、Stats 数据源；Stratz 下：Rank、Period、Position）。
